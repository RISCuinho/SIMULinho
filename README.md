SIMULinho
=========

Simulador integrado iVerilog com interface QT, permite visualizar internamente e controlar a simulação do RISCuinho usando iVerilog.

## Dependências

### QT5

É importante certificar que o pacote qtdeclarative5-dev esteja instalado caso receba a mensagem de erro: `:-1: error: Unknown module(s) in QT: quick`

sudo apt install qtdeclarative5-dev

### iVerilog

o iVerilog deve estar instalado.

sudo apt install iverilog
